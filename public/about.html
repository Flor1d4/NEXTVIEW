About NextView

NextView is a streaming platform developed as part of a student project — a Netflix clone with green branding. This platform allows users to register, login, browse movies and series, and mark favorites. It features popups for login/signup, dynamic content loading using localStorage, and responsive design for modern web.

All media (covers, videos, icons) used in this project are placeholders. The project was built using HTML, CSS, and JavaScript (no frameworks), with a basic Node.js backend for authentication.

Project goal: practice full UI/UX and frontend development with light backend support.
